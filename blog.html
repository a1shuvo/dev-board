Question-1: What are the different ways to select an element in the DOM?

Genereally there are 5 different ways to select an element in the DOM.

  • getElementById(): Select an element by using an unique ID.
  • getElementsByClassName(): Select all elements by using a class name.
  • getElementsByTagName(): Select all elements by using a tag name.
  • querySelector(): Select first element by using CSS selector.
  • querySelectorAll(): Select all elements by using CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent are given below.

  • innerHTML: Returns or sets the HTML content inside an element, including any HTML tags, text content & spacing.
  • innerText: Returns or sets the text content inside an element without CSS hidden text.
  • textContent: Returns or sets the text content inside an element with CSS hidden text.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where we can use a single event listener to a parent element instead of multiple event listeners to its child elements. This technique uses event propagation or bubbling to track & manipulate targeted child elements. It helps to improve performance and handles dynamically added elements as well.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behaviour in the DOM, when a event happens it not only triggers the targeted element but also "bubbled up" to its parent elements until reaches the DOM root element. Like if a click event occurs, the event will propagate upwards to its parent, grandparent & so on.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript there are various methods avaibale to create, add, and remove element in the DOM. Here are given some basic methods for creating, adding, and removing elements in JavaScript.

  • document.createElement(): This method create a new element in the DOM
  • appendChild(): This method add newly created element to the DOM.
  • removeChild(): This method remove an element from the DOM.
